# Stripped CIF Generator

## Overview

The Stripped CIF Generator is a utility module for creating minimal CIF files that contain only the fundamental crystallographic information needed for structural analysis. It removes all metadata, experimental details, and database deposition information, leaving only the essential structural data.

## Location

- **Main module**: `src/utils/cif_stripper.py`
- **Default header**: `src/utils/default_cif_header.txt`
- **GUI integration**: `src/gui/main_window.py` (File menu)

## Purpose

The stripped CIF is designed for:
- Quick structural analysis without database metadata
- Sharing crystallographic data while removing proprietary information
- Creating minimal files for computational studies
- Educational purposes where only core structural data is needed
- Institutional data sharing with custom headers

## What's Included in a Stripped CIF

### Data Block
- Structure name (`data_` block)

### Symmetry Information
- `_symmetry_cell_setting`
- `_symmetry_space_group_name_H-M`
- `_symmetry_Int_Tables_number`
- `_space_group_name_Hall`
- And related alternative field names

### Symmetry Operations (Loop)
- `_symmetry_equiv_pos_site_id`
- `_symmetry_equiv_pos_as_xyz`
- And related alternative field names

### Unit Cell Parameters
- `_cell_length_a`, `_cell_length_b`, `_cell_length_c`
- `_cell_angle_alpha`, `_cell_angle_beta`, `_cell_angle_gamma`
- `_cell_volume`

### Atomic Coordinates (Loop)
- `_atom_site_label`
- `_atom_site_type_symbol`
- `_atom_site_fract_x`, `_atom_site_fract_y`, `_atom_site_fract_z`
- `_atom_site_U_iso_or_equiv`
- `_atom_site_thermal_displace_type`
- `_atom_site_occupancy` (when present)

## What's Removed

All other CIF fields are removed, including but not limited to:
- Experimental conditions
- Crystal preparation details
- Data collection parameters
- Refinement details
- Publication information
- Author information
- Database identifiers
- Equipment details

## Usage

### GUI Usage (Recommended)

1. Open CIF Checker application
2. Select **File → Generate Stripped CIF...** from the menu
3. Choose input CIF file
4. Choose output location
5. Select header type:
   - **Default header**: Professional generic information
   - **Custom header**: Select your own header file
   - **No header**: Clean CIF data only
6. Review generation statistics

### Programmatic Usage

```python
from utils.cif_stripper import StrippedCIFGenerator

# Create generator
generator = StrippedCIFGenerator()

# Generate with different header types
cif_content = "data_example\n_cell_length_a 10.0\n..."

# With default header
stripped_default = generator.generate_stripped_cif(cif_content, header_type='default')

# With custom header
generator.load_custom_header("my_institutional_header.txt")
stripped_custom = generator.generate_stripped_cif(cif_content, header_type='custom')

# Without header
stripped_clean = generator.generate_stripped_cif(cif_content, header_type='none')

# From file to file
success = generator.generate_stripped_cif_from_file(
    "input.cif", 
    "output_stripped.cif",
    header_type='default'
)
```

### Validation

```python
# Check if CIF is suitable for stripping
validation = generator.validate_cif_for_stripping(cif_content)

print(f"Suitable: {validation['is_suitable']}")
print(f"Has coordinates: {validation['has_atomic_coordinates']}")
print(f"Missing fields: {validation['missing_critical_fields']}")
```

### Header System

#### Default Header
- Professional, generic header suitable for any institution
- Explains what a stripped CIF contains
- Located at `src/utils/default_cif_header.txt`

#### Custom Headers
- Users can provide their own header files
- Useful for institutional branding or specific requirements
- Loaded dynamically via `load_custom_header()` method

## Header Examples

### Default Header

The default header file (`default_cif_header.txt`) contains professional, generic information:

```
# Stripped CIF File
# 
# This is a minimal CIF containing only essential crystallographic data:
#   - Crystal structure name
#   - Space group and symmetry information  
#   - Unit cell parameters
#   - Atomic coordinates and displacement parameters
#
# All experimental details, metadata, and publication information
# have been removed to create a focused structural dataset.
#
# Generated by CIF Checker - Stripped CIF Generator
# For more information: https://github.com/danielnrainer/CIF_checker
```

### Custom Headers

Users can create custom header files for institutional branding or specific requirements. Example institutional header:

```
#######################################################################
#
#                 Your Institution Name
#                 Crystallography Department
#
#                Date: 2025-08-26
#
#######################################################################
```

## API Reference

### Class: StrippedCIFGenerator

#### Constructor
```python
StrippedCIFGenerator()
```
Initialize the generator with default header loading.

#### Methods

##### generate_stripped_cif()
```python
generate_stripped_cif(cif_content: str, header_type: str = 'none') -> str
```
Generate a stripped CIF from input content.

**Parameters:**
- `cif_content`: Raw CIF file content
- `header_type`: Header type ('default', 'custom', 'none')

##### load_custom_header()
```python
load_custom_header(header_file_path: str) -> bool
```
Load custom header from file. Returns True if successful.

##### generate_stripped_cif_from_file()
```python
generate_stripped_cif_from_file(input_file_path: str, output_file_path: str, 
                               header_type: str = 'none') -> bool
```
Generate stripped CIF from file and save to another file.

##### validate_cif_for_stripping()
```python
validate_cif_for_stripping(cif_content: str) -> Dict[str, any]
```
Validate if CIF contains sufficient information for meaningful stripping.

Returns:
- `is_suitable`: Boolean indicating if CIF is suitable
- `missing_critical_fields`: List of missing essential fields
- `has_atomic_coordinates`: Boolean for coordinate presence
- `has_symmetry_info`: Boolean for symmetry information presence
- `total_essential_fields_present`: Count of essential fields found
- `recommendations`: List of suggestions

##### get_essential_fields_list()
```python
get_essential_fields_list() -> List[str]
```
Get list of all field names retained in stripped CIFs.

## Integration with Main Application

The stripped CIF generator is designed as a standalone utility that can be easily integrated into the main GUI application at a later stage. The module follows the same patterns as other utilities in the project and uses the existing CIF parser infrastructure.

## Notes

- The generator preserves loop structures for symmetry operations and atomic coordinates
- Field names are case-insensitive when filtering
- Both CIF1 and CIF2 field naming conventions are supported
- Original formatting and multiline structures are preserved where appropriate
- The generator provides validation to ensure the resulting stripped CIF contains meaningful data
